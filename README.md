# Python Test App for Deployment on Cloud Run + Cloud SQL

このリポジトリは、**Cloud Run** と **Cloud SQL** を使用してデプロイするための、Pythonテストアプリケーションです。
This repository is a Python test application designed to be deployed using Cloud Run and Cloud SQL.

## 📦 概要

- Pythonで構築された軽量なテストアプリケーション
- Google Cloud Platform（GCP）の **Cloud Run** でホスティング
- **Cloud SQL（PostgreSQL）** をバックエンドデータベースとして使用
- Cloud SQL Auth Proxy を使用してセキュアに接続

- A lightweight test application built with Python
- Hosted on Cloud Run in Google Cloud Platform (GCP)
- Uses Cloud SQL (PostgreSQL) as the backend database
- Connects securely using the Cloud SQL Auth Proxy

## 🚀 使用技術

- Python 3.x
- Flask
- Cloud Run
- Cloud SQL（例：PostgreSQL）
- Docker
